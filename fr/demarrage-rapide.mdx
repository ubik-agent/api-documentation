## Pour commencer

Ce guide de démarrage rapide vous guidera à travers les étapes essentielles pour effectuer votre premier appel API.

### 1. Obtenez votre clé API

Vous pouvez trouver votre clé API dans les paramètres de votre utilisateur dans l'application.

### 2. Effectuez votre première requête

Voici un exemple de comment effectuer une requête vers le point de terminaison `/users/me` pour obtenir les détails de votre utilisateur.

```bash
curl -X GET "https://app.ubik-agent.com/api/v1/users/me" \
     -H "X-API-KEY: VOTRE_CLE_API"
```

### 3. Prochaines étapes

Maintenant que vous avez effectué votre premier appel API, vous pouvez explorer le reste de notre documentation API pour en savoir plus sur ce que vous pouvez faire.
