Bienvenue dans l'API UBIK ! Ce guide vous accompagnera pour effectuer votre premier appel API en quelques minutes.

<Steps>
<Step title="Obtenez votre clé API">
  Votre clé API est votre passeport pour l'écosystème UBIK. Vous en aurez besoin pour authentifier toutes vos requêtes API. Vous pouvez trouver votre clé API sur la page des paramètres de votre compte dans le tableau de bord UBIK.

  <Card
    title="Accéder aux paramètres de la clé API"
    icon="key"
    href="https://app.ubik-agent.com/user/api-keys"
  />

  <Tip>
    Gardez votre clé API en sécurité comme un mot de passe et ne la partagez pas publiquement.
  </Tip>
</Step>

<Step title="Effectuez Votre Première Requête">
  Vérifions que votre clé API fonctionne correctement en récupérant les détails de votre utilisateur. Nous utiliserons le point de terminaison `/users/me`, qui fournit des informations sur l'utilisateur authentifié.

  <CodeGroup title="GET /users/me">
  ```bash cURL
  curl -X GET "https://app.ubik-agent.com/api/v1/users/me" \
       -H "X-API-KEY: VOTRE_CLE_API"
  ```
  ```python Python
  import requests

  api_key = "VOTRE_CLE_API"
  headers = {
      "X-API-KEY": api_key
  }
  response = requests.get(
      "https://app.ubik-agent.com/api/v1/users/me",
      headers=headers
  )
  print(response.json())
  ```
  ```javascript JavaScript
  const apiKey = "VOTRE_CLE_API";
  const headers = {
    "X-API-KEY": apiKey,
  };

  fetch("https://app.ubik-agent.com/api/v1/users/me", {
    method: "GET",
    headers: headers,
  })
    .then((response) => response.json())
    .then((data) => console.log(data))
    .catch((error) => console.error("Erreur:", error));
  ```
  </CodeGroup>
</Step>

<Step title="Vérifiez la Réponse">
  Si tout est correct, vous recevrez une réponse `200 OK` avec un objet JSON contenant les détails de votre utilisateur.

  ```json Réponse
  {
    "username": "testuser",
    "full_name": "Test User",
    "uuid_id": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
    "created_at": "2025-09-28T12:00:00Z",
    "balance": 100.0,
    "chat_session_count": 5,
    "document_count": 42,
    "workspace_count": 3,
    "assistant_count": 2
  }
  ```
</Step>
</Steps>

Félicitations ! Vous avez effectué avec succès votre premier appel API.
