Bienvenue dans l'API UBIK ! Ce guide vous accompagnera à travers les étapes essentielles pour être rapidement opérationnel. Vous apprendrez à vous authentifier et à effectuer votre première requête pour récupérer les détails de votre utilisateur.

## 1. Obtenez votre clé API

Votre clé API est votre passeport pour l'écosystème UBIK. Vous en aurez besoin pour authentifier toutes vos requêtes API.

Vous pouvez trouver votre clé API sur la page des paramètres de votre compte dans le tableau de bord UBIK.

<Card
  title="Accéder aux paramètres de la clé API"
  href="https://app.ubik-agent.com/user/api-keys"
/>

Gardez votre clé API en sécurité et ne la partagez pas publiquement.

## 2. Effectuez Votre Première Requête

Vérifions que votre clé API fonctionne correctement en récupérant les détails de votre utilisateur. Nous utiliserons le point de terminaison `/users/me`, qui fournit des informations sur l'utilisateur authentifié.

Ouvrez votre terminal et exécutez la commande suivante, en remplaçant `VOTRE_CLE_API` par la clé que vous venez de copier.

```bash
curl -X GET "https://app.ubik-agent.com/api/v1/users/me" \
     -H "X-API-KEY: VOTRE_CLE_API"
```

## 3. Vérifiez Votre Réponse

Si tout est correct, vous recevrez une réponse `200 OK` avec un objet JSON contenant vos informations utilisateur. Cela devrait ressembler à ceci :

```json
{
  "username": "testuser",
  "full_name": "Test User",
  "uuid_id": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
  "created_at": "2025-09-28T12:00:00Z",
  "balance": 100.0,
  "chat_session_count": 5,
  "document_count": 42,
  "workspace_count": 3,
  "assistant_count": 2
}
```

Félicitations ! Vous avez réussi votre premier appel API.
</Check>
