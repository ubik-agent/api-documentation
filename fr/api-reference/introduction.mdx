## Authentification

Tous les points de terminaison de l'API sont authentifiés à l'aide de clés d'API (`X-API-KEY`) et sont définis dans le fichier de spécification.

```json
"securitySchemes": {
    "APIKeyHeader": {
    "type": "apiKey",
    "in": "header",
    "name": "X-API-KEY"
    }
}
```
