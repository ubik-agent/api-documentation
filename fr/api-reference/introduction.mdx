Bienvenue dans la Référence API d'UBIK. Cette documentation fournit des informations détaillées sur tous les points de terminaison, paramètres et réponses disponibles.

Pour des tutoriels plus orientés objectifs et des guides étape par étape, veuillez consulter notre [section Guides](/fr/index).

## Authentification

Toutes les requêtes vers l'API UBIK doivent être authentifiées. L'authentification est gérée en fournissant votre clé d'API secrète dans l'en-tête `X-API-KEY` de chaque requête.

Vous pouvez trouver et gérer vos clés d'API dans le tableau de bord UBIK, dans les paramètres de votre utilisateur.

```bash cURL
-H "X-API-KEY: VOTRE_CLE_API"
```

<Tip>
  Gardez votre clé d'API en sécurité comme s'il s'agissait d'un mot de passe et ne l'exposez dans aucun code côté client.
</Tip>

## Point de Terminaison de l'API

L'URL de base pour tous les points de terminaison de l'API est :

```
https://app.ubik-agent.com/api/v1
```
