## Getting Started

This quickstart guide will walk you through the essential steps to make your first API call.

### 1. Get your API Key

You can find your API key in your user settings in the application.

### 2. Make your first request

Here is an example of how to make a request to the `/users/me` endpoint to get your user details.

```bash
curl -X GET "https://app.ubik-agent.com/api/v1/users/me" \
     -H "X-API-KEY: YOUR_API_KEY"
```

### 3. Next Steps

Now that you've made your first API call, you can explore the rest of our API documentation to learn more about what you can do.
