## How UBIK Works

UBIK's core philosophy is **multistep logic**. Instead of treating AI as an unpredictable black box, we break down complex problems into smaller, manageable tasks. This dramatically improves the reliability, efficiency, and capability of our agents.

Our API gives you programmatic access to UBIK's core components, allowing you to:
-   **Manage Workspaces**: Organize your projects and data in high-level containers.
-   **Ingest Documents**: Feed information into the platform from files or URLs.
-   **Execute Tools**: Leverage powerful, pre-built functions to perform complex tasks.
-   **Run Agent Sessions**: Interact with stateful AI agents that maintain conversation history and context.
-   **Embed Tools**: Display UBIK's UI components directly within your own application.

Ready to start building? Head over to our [Quickstart guide](/en/quickstart) to get started.

## Beyond the API: The UBIK Platform

While this documentation focuses on the API, the full UBIK platform offers a rich web interface for deeper customization of your AI agents. Once you've used the API to manage your foundational resources, you can log in to the UBIK dashboard to:

-   **Define Agent Behavior**: Configure how your agent thinks and processes information through custom logic loops.
-   **Shape Agent Instructions**: Mold your agent's personality, purpose, and response guidelines.
-   **Build Custom Tools Visually**: Use our no-code editor to create and chain tools together.
